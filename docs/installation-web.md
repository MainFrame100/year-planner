# Установка для Claude.ai Web/Desktop

## Требования

- Аккаунт Claude.ai (Pro или Team)
- Браузер или Claude Desktop приложение

## Установка

### Шаг 1: Скачать ZIP

Скачайте `year-compass-web-v1.0.zip` из [Releases](https://github.com/MainFrame100/year-planner/releases).

### Шаг 2: Загрузить в Claude

1. Откройте [claude.ai](https://claude.ai)
2. Перейдите в **Settings** (иконка шестерёнки)
3. Выберите **Capabilities**
4. Найдите раздел **Skills**
5. Нажмите **Upload skill**
6. Выберите скачанный ZIP-файл
7. Включите skill (toggle ON)

### Шаг 3: Создать Project (рекомендуется)

Для сохранения прогресса между сессиями:

1. Перейдите в **Projects**
2. Нажмите **New Project**
3. Назовите: "Годовой план 2026"

## Использование

### Первый запуск

В любом чате напишите:

```
/year-compass
```

или просто:

```
Давай начнём годовое планирование
```

### Выбор фреймворка

Claude предложит:
- **Year Compass** — глубокая эмоциональная работа
- **Navigator** — фокус на 3 приоритетах
- **Гибрид** — комбинация обоих

### Работа с артефактами

После выбора фреймворка Claude создаст артефакт с вашим планом.

**В конце каждой сессии:**
1. Claude напомнит сохранить артефакт
2. Нажмите **Add to Project** под артефактом
3. Выберите ваш Project "Годовой план 2026"

### Продолжение работы

1. Откройте Project "Годовой план 2026"
2. Начните новый чат в контексте Project
3. Claude увидит ваш файл и предложит продолжить

## FAQ

### Почему нужен Project?

Файлы в Project сохраняются между сессиями. Без Project ваш прогресс потеряется после закрытия чата.

### Можно редактировать файл в Project?

Нет, файлы в Project read-only. Claude создаёт новый артефакт каждую сессию, копируя содержимое старого.

### Что если я забыл сохранить?

К сожалению, артефакт будет потерян. Claude напоминает сохранять в конце каждой сессии.

### Как удалить старую версию?

В Project найдите старый файл и нажмите иконку корзины.

## Troubleshooting

### Skill не появился

- Проверьте, что ZIP загрузился без ошибок
- Убедитесь, что toggle включен в Settings → Capabilities

### Claude не видит файл в Project

- Убедитесь, что чат открыт в контексте Project
- Проверьте, что файл `annual_plan_*.md` есть в Project

### Артефакт не создаётся

Попробуйте явно попросить: "Создай артефакт с планом"
