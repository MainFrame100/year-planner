# Примеры заполненных планов

Эти примеры показывают, как может выглядеть результат работы со skill'ом Year Compass.

## Персонаж: Алексей Морозов

**Кто:** 36 лет, Санкт-Петербург, основатель IT-консалтинга "Интегратор Про"

**Бизнес:** Внедрение CRM (amoCRM, Битрикс24), автоматизация бизнес-процессов. 5 лет на рынке, вырос из фриланса.

**Ситуация:** Бизнес в стагнации — доход просел на 25%, команда сократилась с 4 до 2 человек. Начал развивать личный бренд через YouTube и вебинары, но пока не монетизируется.

**Семья:** Жена Марина (33), сын Артём (7), дочь Соня (4).

## Что показывают примеры

- **Глубина проработки** — эмоциональная рефлексия, осознания, благодарности
- **WOOP** — для каждого приоритета (Wish-Outcome-Obstacle-Plan)
- **Implementation Intentions** — конкретные if-then планы
- **Pre-Mortem** — анализ потенциальных провалов заранее
- **Декомпозиция** — от годовых целей до понедельных задач
