# Year Compass - Годовое планирование с Claude

Skill для годового планирования и ретроспективы с использованием доказательных методов (WOOP, Implementation Intentions, Pre-Mortem).

## Зачем это нужно

Планирование года повышает шансы достичь целей — это доказано исследованиями. Есть отличные системы — [Year Compass](https://yearcompass.com/), [Навигатор года](https://t.me/rinatplannerbot) — но заполнять их в одиночку тяжело:

- **Сложно думать долго одному** — мысли разбегаются, нет обратной связи
- **Страх пустой страницы** — не знаешь с чего начать, боишься выбрать "не то"
- **Трудно вспомнить год** — помнишь только яркие моменты, детали размыты
- **Устаёшь раньше, чем заполнишь** — 50 вопросов буклета выматывают

### Мой опыт

Два года я пытался провести детальное планирование — скачивал Year Compass, распечатывал, начинал заполнять... и откладывал. Слишком много вопросов, слишком сложно формулировать в одиночку.

В этом году я создал себе AI-помощника. И **впервые прошёл весь путь до конца**.

Результат: я чётко вижу, что делать в январе, чтобы приблизиться к целям. Не абстрактные "больше читать", а конкретные действия с привязкой к календарю. Плюс теперь есть помощник для ежемесячных review — синхронизация с планом станет проще.

### Решение: AI-помощник

Skill решает проблемы заполнения:

- **Диалог вместо монолога** — AI задаёт уточняющие вопросы, помогает формулировать
- **Голосовой ввод** — говорить быстрее и мысли льются свободнее (очень рекомендую!)
- **Исследования встроены** — типичные ошибки планирования зашиты в skill, AI помогает их избегать

## Как пользоваться

### Шаг 1: Создайте Project (для Web/Desktop)

Рекомендуем вести годовое планирование в отдельном Project:
1. Claude.ai → Projects → New Project
2. Назовите: "Годовой план 2026"
3. Все сессии планирования ведите внутри этого проекта

### Шаг 2: Запуск

**Claude Code:**
```
> /year-compass
```

**Claude.ai Web/Desktop (внутри Project):**
```
Давай начнём годовое планирование, используя Skill year-compass
```

### Шаг 3: Выбор фреймворка

Claude предложит три варианта:

| Фреймворк | Для кого | Фокус |
|-----------|----------|-------|
| **Year Compass** | Любите глубокую рефлексию | Прощение, благодарность, отпускание |
| **Navigator** | Предпочитаете чёткую структуру | 3 приоритета, роли, идентичность |
| **Гибрид** | Хотите всё | Рефлексия + Структура |

### Шаг 4: Заполнение

Claude проведёт вас через процесс:

```
┌─────────────────────────────────────────────────────────────┐
│  СЕССИЯ 1: Ретроспектива (2-3 часа)                         │
├─────────────────────────────────────────────────────────────┤
│  → Обзор месяцев (откройте календарь и фото)               │
│  → Победы и достижения                                      │
│  → Уроки и инсайты                                          │
│  → Карта энергии (что заряжает / сливает)                  │
│  → Благодарность                                            │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  СЕССИЯ 2: Планирование (2-3 часа)                          │
├─────────────────────────────────────────────────────────────┤
│  → Видение на 3-5 лет                                       │
│  → 3 приоритета года (с методом WOOP)                      │
│  → 10 желаний (без обязательств)                           │
│  → Pre-Mortem: что может пойти не так                      │
└─────────────────────────────────────────────────────────────┘
```

### Шаг 5: Ежемесячные обзоры

В первую неделю каждого месяца:
```
> review
```

Claude поможет:
- Проверить прогресс по приоритетам
- Скорректировать цели при необходимости
- Зафиксировать уроки месяца

### Ключевые принципы

- **Один вопрос за раз** — без перегрузки
- **Глубина 3/5 достаточна** — лучше пройти всё, чем застрять на деталях
- **Перерывы проактивно** — после эмоциональных секций
- **Методы вплетены** — не отдельный шаг, а часть процесса

## Советы по использованию

**Не нужно делать всё идеально сразу.** Главный плюс AI — можно возвращаться, уточнять, докручивать. Глубина 3/5 на первом проходе — уже хороший результат.

**Спрашивайте, а не просите заполнить за вас.** Если пункт непонятен — спросите: "Что такое Pre-Mortem?", "Как правильно формулировать Implementation Intentions?". AI объяснит подход и поможет найти варианты. Но заполнять должны вы сами — иначе это будут не ваши цели.

**После заполнения — сделайте review в новом чате.** Загрузите готовый план и попросите AI проанализировать: насколько оптимально подобраны цели, нет ли типичных ошибок планирования. Это даёт обзорную картину и подсвечивает проблемные зоны.

**Используйте для планирования месяца.** Skill можно использовать не только для review, но и для планирования. Попросите помочь распланировать месяц так, чтобы не забыть про каждый приоритет — AI поможет разбить годовые цели на конкретные задачи.

**Поставьте напоминание в календарь** на первую неделю каждого месяца. Открывайте план и обсуждайте прогресс с AI в формате диалога.

**Главное от вас** — регулярность. Skill берёт на себя структуру и методы, но возвращаться к целям — ваша задача.

## Команды

| Команда | Действие |
|---------|----------|
| `/year-compass` | Запустить skill |
| `status` | Показать прогресс заполнения |
| `start [раздел]` | Начать конкретный раздел |
| `break` / `перерыв` | Сделать паузу (прогресс сохранится) |
| `review` | Месячный обзор |
| `shallow` | Показать недопроработанные разделы |
| `deepen [раздел]` | Углубить проработку раздела |

## Подробности

<details>
<summary><b>Научные методы (11 доказательных подходов)</b></summary>

Skill автоматически применяет:

1. **WOOP** (Wish-Outcome-Obstacle-Plan) — для каждого приоритета
2. **Implementation Intentions** (If-Then планы) — конкретные триггеры действий
3. **Pre-Mortem Analysis** — предвидение провалов до их наступления
4. **Reference Class Forecasting** — реалистичные оценки на основе прошлого опыта
5. **Identity-Based Habits** — "Я — человек, который..."
6. **Habit Stacking** — привязка новых привычек к существующим
7. **Environment Design** — изменение окружения для поддержки целей
8. **Minimum Viable Habits** — 2-минутное правило
9. **Progress Tracking** — визуализация прогресса
10. **Accountability** — партнёры по подотчётности
11. **Forgiveness Protocol** — "никогда не пропускай дважды"
