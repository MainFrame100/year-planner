# Year Compass - Годовое планирование с Claude

Структурированный skill для годового планирования и ретроспективы с использованием доказательных методов (WOOP, Implementation Intentions, Pre-Mortem).

## Что это?

Year Compass помогает:
- Провести глубокую ретроспективу уходящего года
- Поставить 3 ключевых приоритета на год
- Применить научно-обоснованные методы постановки целей
- Проводить ежемесячные обзоры прогресса

### Web/Desktop

Для пользователей Claude.ai в браузере или Desktop приложении.

1. Скачайте ZIP из [Releases](../../releases)
2. Откройте Claude.ai → Settings → Capabilities
3. Нажмите "Upload skill" и выберите ZIP
4. Включите skill (toggle ON)

## Как работает

### Фреймворки

- **Year Compass** — глубокая эмоциональная работа (прощение, благодарность)
- **Navigator** — фокус на идентичности (максимум 3 приоритета)
- **Гибрид** — лучшее из обоих миров

### Научные методы

Skill автоматически применяет 11 доказательных методов:

1. **WOOP** (Wish-Outcome-Obstacle-Plan) — для каждого приоритета
2. **Implementation Intentions** (If-Then планы) — конкретные триггеры
3. **Pre-Mortem Analysis** — предвидение провалов
4. **Reference Class Forecasting** — реалистичные оценки
5. **Identity-Based Habits** — "Я — человек, который..."
6. И ещё 6 методов

### Процесс

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ Retrospective│ →   │  Planning   │ →   │ Maintenance │
│  (2-3 часа)  │     │  (2-3 часа)  │     │ (ежемесячно)│
└─────────────┘     └─────────────┘     └─────────────┘
```

**Ретроспектива:** Обзор месяцев → Победы → Уроки → Энергия → Благодарность

**Планирование:** Видение → 3 Приоритета (WOOP) → Желания → Pre-Mortem

**Сопровождение:** Ежемесячные обзоры прогресса и корректировка

## Ключевые принципы

1. **Один вопрос за раз** — без перегрузки
2. **Глубина 3/5 достаточна** — лучше пройти всё, чем застрять на деталях
3. **Перерывы проактивно** — после эмоциональных секций
4. **Методы вплетены** — не отдельный шаг, а часть процесса
