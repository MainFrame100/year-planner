## Философия месячных обзоров

> "Планировщик становится домом для твоих целей" — Navigator

**Почему 80% бросают цели к февралю:**
- Забывают про них (нет напоминаний)
- Нет точек проверки (всё или ничего)
- Первая неудача = полный отказ

**Решение:** Регулярные, структурированные обзоры с встроенным прощением.

---

## Когда проводить

### Оптимальное время
- **Первая неделя месяца** (1-7 число)
- Выходной или вечер буднего дня
- 30-45 минут на полный обзор

### Напоминание
Установи повторяющееся событие в календаре:
```
Что: Месячный обзор года
Когда: 1-е число каждого месяца, 10:00
Напоминание: За 1 день
```

### Минимальный вариант
Если совсем нет времени — хотя бы:
1. Открой план (2 мин)
2. Посмотри на 3 приоритета (2 мин)
3. Оцени: on track / off track (1 мин)
4. Одно действие на неделю (2 мин)

## Процесс корректировки целей

### Когда ИЗМЕНИТЬ цель

**Признаки:**
- Внешние обстоятельства существенно изменились
- Появилась новая информация
- Нашёлся лучший путь к тому же результату

**Процесс:**
1. Сформулируй новую версию цели
2. Проверь: это тот же результат, другой путь?
3. Обнови Implementation Intentions
4. Запиши причину изменения (для истории)

### Когда ОТЛОЖИТЬ цель

**Признаки:**
- Временно нет ресурсов (время, деньги, энергия)
- Другой приоритет объективно важнее сейчас
- Внешние зависимости (жду кого-то/что-то)

**Процесс:**
1. Определи условие возврата ("Вернусь, когда...")
2. Поставь напоминание проверить условие
3. Не удаляй из плана — пометь как "отложено"

### Когда УБРАТЬ цель

**Признаки:**
- Больше не соответствует ценностям/видению
- Альтернативная стоимость слишком высока
- Заменена более важным приоритетом

**Процесс:**
1. Признай: это осознанное решение, не провал
2. Запиши урок: что эта цель дала/показала
3. Освободи место (ментально и в плане)

### Когда ПРОДОЛЖАТЬ несмотря ни на что

**Признаки:**
- Находишься в "дипе" (середина пути)
- Нет фундаментальных причин менять
- Прогресс есть, просто медленнее ожиданий

**Процесс:**
1. Перечитай своё "Почему" (раздел Смысл)
2. Найди маленький следующий шаг
3. Обратись к accountability partner
4. Примени Forgiveness Protocol

## 5 стадий достижения цели

Понимание психологического пути помогает не сдаваться в трудные моменты.

```
                    ★ Достижение
                   /
                  /
      Осознанный /
      оптимизм  /
               /
──────────────/──────────────────── Линия "нормы"
             /
            /
    Кризис ●  ← "Забей" (опасная точка)
           \
            \
     Пессимизм от реальности
              \
               \
    Оптимизм ●  ← Старт (незнание)
    от незнания
```

### Стадия 1: Оптимизм от незнания
**Когда:** Начало, первые недели
**Ощущение:** "Это будет легко!"
**Риск:** Недооценка сложности

### Стадия 2: Пессимизм от реальности
**Когда:** Через 2-4 недели
**Ощущение:** "Это сложнее, чем думал"
**Риск:** Начать сомневаться

### Стадия 3: Кризис ("Забей")
**Когда:** 1-2 месяца
**Ощущение:** "Может, это не для меня"
**Риск:** Бросить (80% здесь сдаются)

### Стадия 4: Осознанный оптимизм
**Когда:** После преодоления кризиса
**Ощущение:** "Я понимаю как это работает"
**Признак:** Видишь прогресс, знаешь следующие шаги

### Стадия 5: Достижение
**Когда:** Финал
**Ощущение:** "Я сделал это"
**Важно:** Отпраздновать!

## Troubleshooting

### "Я пропустил месячный обзор"

**Не паникуй.** Сделай короткую версию прямо сейчас:
1. Где я сейчас по каждому приоритету? (3 мин)
2. Что мешает? (2 мин)
3. Один шаг на эту неделю (2 мин)

### "Я отстаю по всем целям"

**Возможные причины:**
- Слишком много целей → сократи до 1-2
- Цели слишком амбициозные → уменьши scope
- Нет системы → добавь Implementation Intentions
- Нет энергии → проверь sleep/nutrition/stress

### "Я потерял мотивацию"

**Попробуй:**
1. Перечитай раздел "Смысл" для приоритета
2. Вспомни свою роль ("Я — ...")
3. Сделай самый маленький шаг (2 минуты)
4. Поговори с accountability partner
5. Проверь: может, цель больше не актуальна?

### "Мне стыдно смотреть на план"

**Это нормально.** Стыд — сигнал, не приговор.

1. Примени self-compassion: "Что бы я сказал другу?"
2. Посмотри что ПОЛУЧИЛОСЬ, даже маленькое
3. Переформулируй: "Я учусь планировать" вместо "Я провалился"
4. Используй Forgiveness Protocol
5. Начни с чистого листа на этот месяц

### "Появились новые важные цели"

**Процесс:**
1. Запиши их в раздел "Желания" (не приоритеты)
2. Подожди до следующего обзора
3. Если через месяц всё ещё важно — рассмотри замену
4. Помни: max 3 приоритета одновременно

## Quarterly Deep Review (опционально)

Раз в квартал (март, июнь, сентябрь) проведи расширенный обзор:

1. **Все месячные обзоры** — перечитай, найди паттерны
2. **Видение 3-5 лет** — всё ещё актуально?
3. **3 приоритета** — нужна ли ротация?
4. **Pre-Mortem** — обнови риски
5. **Окружение** — с кем провёл время?
6. **Энергия** — что consistently даёт/забирает?

Время: 60-90 минут
